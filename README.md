# Lab 41 - OAuth

#### Description: 
An application thats let's users log into their github utilizing Oauth.

#### Technologies Used:
1. Express
1. Cookies

#### Tutorial: 
1. Clone repo
1. Run `npm install`
1. run `nodemon`
1. Navigate to ``http://localhost:3000``
1. Complete the steps provided on the app.
